# 🔗 URL Shortener

A simple Flask-based web application that converts long URLs into short, shareable links.

## Features

- Shortens long URLs
- Redirects short URLs to the original website
- Stores URLs using SQLite
- Clean and responsive user interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Project Structure

```
CodeAlpha_URLShortener/
│── app.py
│── requirements.txt
│── templates/
│   └── index.html
│── static/
│   └── style.css
│── README.md
```

## How to Run

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
